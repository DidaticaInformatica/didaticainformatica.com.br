## Como atualizar no GitHub Pages

1. Extraia o ZIP.
2. Envie para o repositório:
   - index.html
   - styles.css
   - script.js
   - robots.txt
   - sitemap.xml
   - manifest.webmanifest
   - README.md
   - assets/
3. Faça commit.
4. Aguarde o GitHub Pages atualizar.
5. Teste em aba anônima ou use Ctrl + F5.
